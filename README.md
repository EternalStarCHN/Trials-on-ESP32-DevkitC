# 介绍
本仓库用于存储基于ESP32-WROOM-32E制作的DevkitC开发板上进行测试的程序，开发板的PCB可参考LiiGuang发布在OSHWHUB上的39脚ESP32通用开发板👉[链接](https://oshwhub.com/LiiGuang/esp32-wroom-32d-typec_copy_copy)。  
开发环境使用VSCode+PlatformIO，基于Arduino的众多库函数进行开发。
